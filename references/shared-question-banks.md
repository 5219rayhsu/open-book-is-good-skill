# 跨考試共用題庫：先找共用槓桿，再開新考試

> 適用 Phase 1（偵察）與 Phase 4（題庫）。**開一套新考試前先做這步**，否則會把同一科目
> 重複建兩遍、還可能兩份不一致。

## 為什麼：國考常有共用基礎科目

升學端（學測／會考）各科彼此不重疊，**預設不跨考混題**。但**國考不同**——同一類專業的
不同考試常**共用基礎科目**，題目甚至出自同一命題機關、同年同卷。

**實例（臨床／諮商心理師）**：兩者同屬專技高考心理師類，基礎科目大量重疊——
**變態心理學、心理衡鑑（心理測驗）、心理學研究法、人類發展**等，clinical 與 counseling
兩組都考。若各做一份：①工時翻倍 ②兩份題庫易漂移（同題兩種詳解、改了一邊忘了另一邊）。

**原則：重疊科目建一次、兩考試共用、題庫層去重乾淨。**

## 怎麼找：開新考試前做「科目重疊矩陣」

1. 列出目標考試集合各自的**應試科目**（從考選部簡章／應考須知，屬 Phase 1 偵察的一部分）。
2. 兩兩比對科目，標出**重疊**。注意陷阱：
   - **同科不同名**：變態心理學 ≈ 心理病理學；心理衡鑑 ≈ 心理測驗。**看內容不看名稱**。
   - **同名不同命題**：不同考試的同名科目若**命題機關不同**，題目其實不同 → **不可合併**。
     合併的前提是「**同一份官方試題**」（同機關、同年、同卷），不是「科目名稱一樣」。
3. 產出矩陣（哪些科目被哪些考試共用）→ 決定哪些建成**共用單元**。

```
            變態心理 心理衡鑑 研究法 諮商理論 心理病理…
臨床心理師     ✓       ✓      ✓               ✓
諮商心理師     ✓       ✓      ✓      ✓
→ 共用：變態心理、心理衡鑑、研究法（建一次，兩考試引用）
→ 專屬：諮商理論(諮商)、心理病理進階(臨床)…各自建
```

## 怎麼建模：題庫層去重 + 多考試標記

共用科目**只存一份題庫**，用「這題屬於哪些考試」的標記表達歸屬，而非複製：

- **schema 擴充**：升學端每題單一 `exam`/`subject`；國考端把 `exam` 改成 **`exams` 陣列**
  （或加 `categories`），例：
  ```json
  { "qid": "心理師_變態心理_2023_12", "subject": "變態心理學",
    "exams": ["clinical", "counseling"], "year": 2023, ... }
  ```
- **qid 用「科目＋年＋題號」**（不綁單一考試），讓同題天然唯一、不會因兩考試各編一次而重複。
- **前端考試選擇器**：選 clinical 或 counseling 時，各自 `filter` 出 `exams` 含該考試的題；
  共用題自然同時出現在兩邊，**底層仍是同一份資料**（沿用「資料驅動、關注點分離」）。
- **詳解同理**：共用題的詳解只寫一份（以 qid 為鍵），兩考試共享 → 永遠一致。

## 冪等與去重紀律

- merge 仍以 **qid 為鍵冪等**（同 source 同題重跑只覆蓋，不疊加）。
- 合併兩考試的同一份試題時，**確認是同一官方卷**（同機關、年、卷）才把對方 exam 加進 `exams`；
  有疑慮就分開存、別亂併（寧可漏、不可錯，同 §4 門檻精神）。
- 開新考試先查重疊矩陣 → 共用科目**先檢查題庫是否已有**（別重抓重解）→ 只補專屬科目。

## 混卷模擬考的內容指紋去重

上面談的是「建置時」的去重；**組卷時**還有一層不同的重複要處理。國考同一份考古題常被**不同年份、不同題號重複收錄**（歷屆援用同一題），混卷模擬考／完整診斷跨年跨卷抽題時，若只認 `qid` 去重，同一題會用兩個不同 `qid` 各自被抽中，考生在同一份模擬卷裡遇到兩題內容完全相同的題目。

**做法：組卷前做內容指紋去重，而非只認 qid。**

- **內容指紋** = 題幹（去除所有空白）＋ 選項（去除空白後**排序**）組成的字串。選項要排序後再組指紋，是因為同一題在不同年份收錄時選項順序可能被重新編排，指紋要比對「內容集合」相同，不是「字面順序」相同。
- 抽題組卷時同指紋只留一筆（優先留新年份），其餘視為重複跳過，換下一題遞補。
- **短題幹護欄**：去空白後題幹長度低於安全門檻（實務上抓在 8 字，涵蓋詞彙型單題、克漏字這類先天短題幹的合法題型）者，**不受這道去重規則影響**，一律保留——否則會把大量合法的短題型誤判成重複而濫殺。

## 同題幹孿生題：絕不可 stem-only 去重

跨年度的「孿生題」——**題幹幾乎相同、但選項完全不同**——在題庫裡並不少見。這種題**技術上是不同題**（換了年份、換了選項組合、常常也換了正解），內容指紋（題幹＋選項）天然不會誤判成重複；但如果為了省事改用**只比對題幹**的去重規則，會把這些真正不同的題錯殺掉。

**鐵律：去重規則永遠要連選項一起比對，孿生題是真不同的題，不能只看題幹刪題。**

但同場互斥另有需求：考生若同一場模擬考同時抽到「題幹幾乎一樣、選項卻不同」的兩題，體感就是遇到重複題，觀感很差——這需要一道**內容指紋擋不住、必須另立的規則**：

- **另立「題幹指紋」**（只取去空白後的題幹本身，不含選項），只在抽題時做「同一場考試不得出現兩題同題幹指紋」的互斥判斷——用途是**在同一場裡挑其中一題、換掉另一題**，不是拿來從題庫刪題。
- **通用萬用題幹必須豁免**，否則會誤傷合法情形：真實考卷本就常見多題共用同一句泛用問法（例如「下列敘述何者錯誤？」僅 10 字），這種題幹本身不構成「感覺像重複」。做法是設一個**正規化後字數門檻**（實務上抓在 <13 字），低於門檻視為泛用問法、不參與同場互斥判斷；只有長度超過門檻、確實是「同一題幹、不同選項」的孿生題，才觸發互斥。
  - 門檻怎麼抓：13 字是取在「已知的通用萬用題幹長度（10 字）」與「已知最短的實質孿生題幹長度（實測 18 字）」兩個邊界之間的一個安全切點，兩層都留出安全餘裕，不卡在邊界值上。
  - 極短或空題幹（如詞彙型單題，門檻同上一節的 8 字護欄）同樣不參與互斥判斷。

一句話：**內容指紋（題幹＋選項）決定「是不是真的同一題」，題幹指紋只決定「同一場要不要都出現」——兩者職責不同，別互相替代。**

## 一句話

**國考各考試對使用者各自獨立呈現，但建置時先找跨考試共用科目、共用題庫一次建好、用 `exams`
陣列標歸屬、題庫層去重** —— 省工又保證一致。
