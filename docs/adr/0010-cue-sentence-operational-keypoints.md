# ADR 0010：題眼句與判準式考點

日期：2026-07-23　狀態：已採納

## 決策

1. **題眼句**：詳解正解段開頭一句先點出「題幹的哪個關鍵詞／條件觸發哪條規則」（計算題為「為何選用此公式」）。
2. **判準式考點**：末段考點的每個概念，一句話從「它是什麼」（定義重述）改為「怎麼判／何時用」的操作句（例：「無權代理——效力未定，相對人得催告本人承認」）。
3. **直覺句**：選用不設義務——最強誘答的誤解若源於不知規範目的，得加一句立法目的／原理。

## 原因

- 考生實戰面對的是題幹；「從題幹訊號觸發規則」是專家早已自動化、詳解卻最少教的一步（專家知識隱形化），題眼句把它顯性化，練習才對得上真實情境。
- 判準式考點是可遷移的提取線索——考生下次遇到新題可直接套用；定義重述只是重讀式回饋，記憶效益低。
- 直覺句不設硬義務：學習科學條款一旦寫成逐項義務，紅隊會系統性過度觸發、詳解冗長（見 ADR 0009 補遺的實測教訓），故僅在誤解確實源於不知規範目的時使用。

## 生效邊界

新規則自公告批次起生效、不回溯舊批；紅隊對題眼句的判準收斂為「完全未連結題幹、背書式給答案」才記低嚴重度。
