# 管線範本示範：操作卡・金樣詳解・錨定素材包

這份文件給共編三則範本，示範詳解生成管線各環節的產物長什麼樣：

- **操作卡**塞進生成 agent 的恆定前綴，告訴它「要寫成什麼樣」（ADR-0005 槓桿 2）。
- **金樣詳解**是同前綴裡的 1 則 few-shot 示範，告訴它「好的成品長這樣」（ADR-0008）。
- **錨定素材包片段**也在前綴裡，讓它「抄現行條文而非憑記憶」（ADR-0005 槓桿 1、ADR-0008）。

操作卡與金樣是一組（皆取自醫師國考詳解線）；素材包片段另取一個法規科的例子，因為法規科的
錨定素材正是「現行條文全文」，最能示範素材包的樣子。三則對照著看，就能拼出生成前綴的全貌。

---

## 一、操作卡範例（醫師國考選擇題，蒸餾版）

> 一張蒸餾操作卡，取代整份規範進 prompt。只留生成當下必須遵守的硬規則，完整理由在對應 ADR，
> 不塞進卡裡稀釋遵循率。醫師線的輸出多帶一個 `ans`（自行推得的正解字母），供答案錨交叉核對。

```markdown
# doctor 詳解操作卡（醫師國考選擇題）

輸出：JSON `{ans, t, c}`。`ans`＝你推理出的正解字母（照實填，勿為對答案而竄改）；`c`＝把握度 high｜med｜low。

**三段結構**（段間空一行、順序固定）
- `正解：`先講正解為何對（計算題逐步列數值），自成一段。
- 中段：每個非正解選項各自成段，字母加全形冒號起頭（`A：`）講為何錯；負向題（何者錯誤／最不適當）則反講此敘述為何對、故非答案。
- `本題考點：`末段，列 1–3 概念（附英文原名）一句話點出。

**硬規則**
- 選項用大寫 A–E、禁數字編號；每個非正解字母在中段恰一段。
- 合併解析：僅同因（同一判準／法規／概念錯位）致錯者可併段，段首頓號相連（`A、C：`），段內仍逐一點出各選項差異；理由不同只結論同錯者分開。
- 禁鷹架詞（更正／經查／如上所述）與免責收尾（以官方答案為準／信心設為低）；爭議題具體點出爭點。
- 不杜撰法條號／劑量／數據／指引名稱，寧缺勿造。
- 繁中臺灣全形標點，數字／單位／英文半形，無簡體字。
- 概念題 120–600 字為度。

**`c` 判準**：high＝教科書級定論；med＝主體對但有次要不確定；low＝機轉／數據不確定、指引時效敏感、查證不足——文中須明講（100% 送紅隊）。

**醫學特注**
- 劑量／單位／時間窗／百分比務求精確；任一數值沒把握即 `c=low` 並文中點明。
- 指引／診斷準則寫名稱、不杜撰版本號或年份；時效敏感標 low。
- 高危題型（劑量／急救／禁忌症）從嚴，寧 low 不裝懂。
- 辨析誘答講「為何看似對卻錯」，不只寫「屬 X 故選」。
```

## 二、金樣詳解範例（醫師國考，完整、自足）

> 一則實際用作 few-shot 金樣的醫師國考詳解：三段結構、逐選項辨析、考點段齊全，且不依賴附圖。
> 金樣入 prompt 前一定先過紅隊（ADR-0008）——範本自己埋錯會系統性汙染同主題生成。

**題目大意**：給一組肺功能檢查數據（FEV1/FVC 為 58%、DLCO 為預測值的 55%），問最可能的診斷。
選項為 (A) 氣喘、(B) 肺血管栓塞、(C) 肺氣腫、(D) 肺纖維化。官方答案：**C**。把握度 `c`：high。

**金樣詳解（`t` 欄內容）**：

```text
正解：先判讀肺功能型態。FEV1/FVC 為 58%，低於 70% 的判準，代表吐氣氣流受阻，屬「阻塞型」通氣障礙；再看瀰散量 DLCO 只有預測值的 55%，明顯下降。阻塞型通氣障礙合併 DLCO 顯著下降，最符合肺氣腫（emphysema）——肺氣腫破壞肺泡壁及其中的微血管，使氣體交換的表面積減少，DLCO 因而降低，故選 C。

A：氣喘也是阻塞型（FEV1/FVC 下降），單看比值容易與肺氣腫混淆；但氣喘的病灶是可逆的支氣管收縮，肺泡與微血管膜完整，DLCO 通常正常甚至偏高，與本題 DLCO 明顯下降不符，是本題最需要靠 DLCO 來鑑別的誘答。

B：肺血管栓塞的肺量計檢查（spirometry）多半正常、不會呈現 FEV1/FVC 下降的阻塞型態；雖然大範圍栓塞可能使 DLCO 下降，但無法解釋題目的阻塞型通氣障礙，故不符。

D：肺纖維化屬「限制型」通氣障礙，特徵是肺容積縮小、FEV1 與 FVC 同步下降而 FEV1/FVC 比值正常或偏高；本題 FEV1/FVC 只有 58%（下降）並非限制型表現，型態不符，故非肺纖維化（其 DLCO 雖也會下降，但通氣型態對不上）。

本題考點：肺功能判讀（pulmonary function test）——FEV1/FVC 區分阻塞型（<70%）與限制型（正常或上升）；瀰散量 DLCO 反映肺泡-微血管的氣體交換面積，用以在阻塞型中鑑別肺氣腫（DLCO 下降）與氣喘（DLCO 正常或上升）。
```

對照三段結構怎麼落地：**正解段**先立型態判讀再導出答案；**選項辨析段** A／B／D 各自成段、
以字母加全形冒號起頭、逐一講「為何看似對卻錯」（尤其 A 點名它是「最需要靠 DLCO 鑑別的誘答」）；
**本題考點段**收束成一句可掃描的重點。這正是 ADR-0003 三段固定結構的成品樣貌。

## 三、錨定素材包片段（現行法條全文）

> 錨定素材包把該科現行條文全文做成整科共用的常數包，塞進生成前綴，讓模型「抄」而非「憑記憶」。
> 條文是政府開放資料、公共領域，直接收錄即可。每筆帶版本戳記，供時效閘比對修法（ADR-0005 槓桿 1）。
> 下面示範一筆社會救助法的條文，說明素材包的格式。

```text
【法規】社會救助法
【版本戳記】現行條文（引用前以全國法規資料庫最新公布版本核對）
【第 4 條】
  本法所稱低收入戶，指經申請戶籍所在地直轄市、縣（市）主管機關審核認定，符合家庭總收入
  平均分配全家人口，每人每月在最低生活費以下，且家庭財產未超過中央、直轄市主管機關公告
  之當年度一定金額者。
  前項最低生活費，由中央、直轄市主管機關參照中央主計機關所公布當地區最近一年每人可支配
  所得中位數百分之六十定之，並於新年度計算出之數額較原公告數額為高時，即應公告調整。
  ……（同條後續各項依現行條文全文收錄，此處示前二項）
```

> 用法：法規科生成 agent 引用某條時，直接抄素材包這段文字；法條引用核對器（零 token）再抽詳解裡的
> 「法名＋條號＋關鍵數字」回頭比對這份全文——條號不存在就攔、關鍵數字（如「百分之六十」）對不上就 flag。
> 這就是「檢索錨定防事實錯」那一層的具體運作（ADR-0008）。醫學科的錨定素材則是診斷準則／指引名稱
> 這類權威來源（不抄受著作權保護的教科書內文），格式相同、來源不同。
